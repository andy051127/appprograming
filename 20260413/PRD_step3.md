# PRD Step 3 - 즐겨찾기 & 저장 기능

## 개요
사용자가 마음에 드는 레시피를 저장하고, 저장된 레시피를 언제든지 다시 확인할 수 있는 즐겨찾기 기능

---

## 목표
- 레시피를 즐겨찾기에 추가/제거
- 저장된 레시피 목록 페이지 제공
- 저장된 레시피의 상세 조리법 재열람 가능
- 데이터는 로컬 파일(JSON) 또는 SQLite DB에 영구 저장

---

## 사용자 스토리
- 사용자는 레시피 조리법 화면에서 즐겨찾기 버튼을 눌러 저장할 수 있다
- 사용자는 저장된 레시피 목록 페이지에서 모든 즐겨찾기를 볼 수 있다
- 사용자는 저장된 레시피를 클릭해 조리법을 다시 확인할 수 있다
- 사용자는 더 이상 필요 없는 즐겨찾기를 삭제할 수 있다
- 사용자는 저장된 레시피를 이름으로 검색할 수 있다

---

## 기술 스택
| 구분 | 기술 |
|------|------|
| Frontend | HTML, CSS, JavaScript |
| Backend | Python Flask |
| 데이터 저장 | SQLite (sqlite3) |
| ORM | 없음 (직접 SQL 또는 SQLite3 모듈 사용) |

---

## 데이터 모델

### favorites 테이블 (SQLite)
| 컬럼명 | 타입 | 설명 |
|--------|------|------|
| id | INTEGER PRIMARY KEY | 자동 증가 ID |
| recipe_name | TEXT NOT NULL | 레시피 이름 |
| ingredients | TEXT | 사용 재료 (JSON 문자열) |
| steps | TEXT | 조리 순서 (JSON 문자열) |
| tips | TEXT | 요리 팁 |
| cooking_time | TEXT | 조리 시간 |
| difficulty | TEXT | 난이도 |
| saved_at | DATETIME | 저장 일시 |

---

## API 명세

### POST /api/favorites
레시피 즐겨찾기 추가

**Request**
```json
{
  "recipe_name": "두부 달걀 볶음",
  "ingredients": ["달걀", "두부", "양파"],
  "steps": ["두부를 자른다.", "볶는다."],
  "tips": "물기를 제거하면 바삭합니다.",
  "cooking_time": "15분",
  "difficulty": "쉬움"
}
```

**Response**
```json
{
  "message": "즐겨찾기에 저장되었습니다.",
  "id": 1
}
```

---

### GET /api/favorites
저장된 즐겨찾기 목록 반환

**Response**
```json
{
  "favorites": [
    {
      "id": 1,
      "recipe_name": "두부 달걀 볶음",
      "cooking_time": "15분",
      "difficulty": "쉬움",
      "saved_at": "2026-04-13 10:30:00"
    }
  ]
}
```

---

### GET /api/favorites/{id}
특정 즐겨찾기 상세 조회

**Response**
```json
{
  "id": 1,
  "recipe_name": "두부 달걀 볶음",
  "ingredients": ["달걀", "두부", "양파"],
  "steps": ["두부를 자른다.", "볶는다."],
  "tips": "물기를 제거하면 바삭합니다.",
  "cooking_time": "15분",
  "difficulty": "쉬움",
  "saved_at": "2026-04-13 10:30:00"
}
```

---

### DELETE /api/favorites/{id}
즐겨찾기 삭제

**Response**
```json
{
  "message": "즐겨찾기가 삭제되었습니다."
}
```

## UI 구성

### 즐겨찾기 버튼 (레시피 상세 화면)
```
+------------------------------------------+
|   두부 달걀 볶음       [ ⭐ 즐겨찾기 추가 ] |
|   1. 두부를 먹기 좋은 크기로 자른다.       |
|   2. 양파를 채 썰어 준비한다.              |
|   ...                                    |
+------------------------------------------+
```

### 즐겨찾기 목록 페이지 (/favorites)
```
+------------------------------------------+
|   ⭐ 저장된 레시피                         |
|   [ 검색창: 레시피 이름 검색 ]             |
+------------------------------------------+
|  +----------------+  +----------------+  |
|  | ⭐ 두부 달걀   |  | ⭐ 볶음밥       |  |
|  |    볶음        |  |                 |  |
|  | ⏱ 15분 ⭐쉬움  |  | ⏱ 20분 ⭐보통  |  |
|  | 저장: 04/13    |  | 저장: 04/13    |  |
|  | [보기] [삭제]  |  | [보기] [삭제]  |  |
|  +----------------+  +----------------+  |
+------------------------------------------+
```

## 기능 상세

### 1. 즐겨찾기 추가
- 레시피 상세 화면의 ⭐ 버튼 클릭 시 저장
- 이미 저장된 레시피는 버튼이 "저장됨" 상태로 표시
- 저장 성공 시 토스트 알림 표시 ("즐겨찾기에 저장되었습니다!")

### 2. 즐겨찾기 목록
- 저장된 레시피를 카드 형태로 표시
- 최신 저장 순서로 정렬
- 레시피 이름 검색 기능 (클라이언트 사이드 필터링)

### 3. 즐겨찾기 상세 보기
- 카드 클릭 시 저장된 조리법 전체 표시
- API 재호출 없이 DB에서 불러옴

### 4. 즐겨찾기 삭제
- 삭제 버튼 클릭 시 확인 다이얼로그 표시
- 확인 후 DB에서 삭제, 목록 즉시 갱신
