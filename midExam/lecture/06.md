## API의 개념과 활용 방법 [cite: 1710-1711]
### API란? [cite: 1712]
API(Application Programming Interface)는 **프로그램과 AI 모델을 이어주는 다리**입니다.
* 마치 전화로 전문가에게 도움을 요청하듯, 프로그램이 AI 모델에게 사진을 분석해 달라고 요청하면 AI가 이미지를 해석하고 결과를 돌려줍니다. [cite: 1713]
* 웹 애플리케이션은 사용자가 업로드할 인터페이스만 제공하고, 핵심 AI 기능은 API가 처리합니다. [cite: 1714]


### API 활용 예시: '냉장고를 부탁해' [cite: 1715-1716]
* 사용자 → [냉장고 사진 업로드] → 앱 → [API] → AI 모델(이미지 인식)
* 사용자 ← [레시피 추천 표시] ← 앱 ← [API] ← AI 모델(텍스트 생성)

### 오픈라우터(OpenRouter)란? [cite: 1722-1726]
* 여러 AI 모델 API를 한곳에 모아 제공하는 플랫폼입니다.
* 클로드, GPT, Llama 등 다양한 모델을 **하나의 API 키**로 사용 가능합니다.
* 무료 모델이 많아 학습 목적으로 활용하기 좋습니다.

## API 키란? (교재 p.227) [cite: 1729-1733]
API 키(API key)는 API를 사용할 때 사용자를 인증하는 **비밀 열쇠**입니다.
* 서버는 API 키를 통해 누가, 얼마나 요청을 보냈는지 확인합니다.
* 키가 없으면 데이터 유출 위험이 있고 사용량 제한이 불가능합니다. [cite: 1734-1736]

### API 키 보안 규칙 [cite: 1737-1740]
1. API 키를 코드에 직접 하드코딩하지 마세요. (X)
2. **`.env`** 파일에 저장하고, **`.gitignore`**에 등록하여 Git에 올리지 마세요.
3. 공유 폴더가 아닌 개인 로컬 컴퓨터에만 보관하세요.

## 단계별 계획 수립하기 (교재 p.244) [cite: 1832]
### 3단계 PRD 분해 전략 [cite: 1833]
[다이어그램: PRD_step1(이미지 인식) → PRD_step2(레시피 생성) → PRD_step3(사용자 프로필) 순으로 이어지는 개발 로드맵] [cite: 1834-1836]

| PRD | 핵심 기능 | AI 모델 |
| :--- | :--- | :--- |
| **Step 1** | Streamlit 기반 웹 UI + 냉장고 재료 인식 | Gemma 3 27B-it |
| **Step 2** | 레시피 생성 + 재료 편집 + 필터링 및 검색 | Qwen 3.6 Plus |
| **Step 3** | 사용자 인증 + 레시피 저장 + 대시보드 | - |

* **컴퓨팅 사고 연결:** 전체 앱을 한 번에 만들지 않고 3단계로 **분해**하여 점진적으로 구현합니다. [cite: 1838] 각 단계는 이전 단계를 기반으로 발전합니다. [cite: 1839]

## 핵심 키워드 정리
### 06-1 이론 키워드 (교재 p.242) [cite: 2010-2011]
* **API(Application Programming Interface):** 외부 AI 서비스와 프로그램을 연결하는 다리로, 간단한 요청만으로 복잡한 AI 기능을 활용할 수 있게 해 주는 통신 규칙 [cite: 2012]
* **API 통합 플랫폼:** 여러 AI 모델을 한 곳에서 비교하고 선택할 수 있는 서비스 (예: 오픈라우터) [cite: 2013]
* **API 키(API key):** API 서비스에 접근하기 위해 발급받아 사용자를 인증하는 비밀 열쇠 [cite: 2014]
* **.env:** API 키가 저장된 환경 변수 파일로, API를 호출할 때마다 참조됨 [cite: 2015]
* **.gitignore:** .env 파일이 자동으로 깃허브에 업로드되지 않도록 특정 파일을 제외하여 보호하는 설정 파일 [cite: 2016]

### 06-2 실습 키워드 (교재 p.255) [cite: 2018-2020]
* **다중 AI 모델 연동:** 이미지 인식, 텍스트 생성 등 목적에 맞게 여러 AI 모델을 활용하는 방식 [cite: 2021]
* **웹 프레임워크:** 웹 애플리케이션을 쉽게 만들 수 있도록 도와주는 도구 모음 (Streamlit 등) [cite: 2022]
