## 클로드 코드란? [cite: 515-519]
클로드 코드(Claude Code)는 터미널(명령 프롬프트)에서 실행되는 AI 코딩 도구입니다.

### 클로드 웹 vs 클로드 코드 [cite: 517-518]
| 구분 | 클로드 웹 (claude.ai) | 클로드 코드 (Claude Code) |
| :--- | :--- | :--- |
| **실행 환경** | 웹 브라우저 | 내 컴퓨터의 터미널 |
| **작업 방식** | Artifacts에서 미리보기 | 실제 파일 생성·실행 |
| **활용 범위** | 간단한 코드, 문서, 웹페이지 | 복잡한 프로그램, 프로젝트 관리 |
| **파일 접근** | 제한적 | 내 컴퓨터 파일에 직접 접근 |

---

## 터미널이란? [cite: 521-525]
텍스트 명령어로 컴퓨터를 조작하는 도구입니다.

### Windows에서 터미널 실행
1. 윈도우 검색창에 `cmd` 입력 → Enter
2. 또는 `Windows 키 + R` → `cmd` 입력 → 확인

### 기본 터미널 명령어 [cite: 526-527]
| 명령어 | 기능 | 예시 |
| :--- | :--- | :--- |
| **dir** | 현재 폴더의 파일 목록 | `dir` |
| **cd** | 폴더 이동 | `cd VibeCoding` |
| **mkdir** | 새 폴더 만들기 | `mkdir Study-01` |

## 실습 ② 손글씨 인식 - MNIST란? [cite: 602-609]
* **OCR (Optical Character Recognition):** 이미지 속 글자를 디지털 텍스트로 변환하는 기술
* **MNIST 데이터셋:**
    * 용도: 숫자(0~9) 손글씨 인식 학습에 사용
    * 데이터: $28 \times 28$ 픽셀 흑백 이미지 7만 장
    * 특징: 딥러닝 입문자들의 표준 데이터셋

[다이어그램: VibeCoding/ 폴더 아래 루트 CLAUDE.md와 각 하위 폴더(desktop_version, web_version)별 CLAUDE.md가 존재하는 구조] [cite: 709-720]

* **루트 CLAUDE.md:** 프로젝트 전체에 적용되는 공통 규칙 [cite: 721]
* **하위 폴더 CLAUDE.md:** 해당 폴더에만 적용되는 특화 규칙 [cite: 722]

## 실습 중 자주 발생하는 문제 [cite: 752]

| **/init 후 파일 없음** | 폴더 권한 또는 경로 문제 | 올바른 폴더에서 재실행 [cite: 753] |

## 핵심 키워드 정리
### 이론 키워드 [cite: 755]
* **터미널(Terminal):** 텍스트 명령어로 컴퓨터를 조작하는 도구 [cite: 756]
* **클로드 코드(Claude Code):** 터미널에서 실행되는 AI 코딩 도구 [cite: 757]
* **OCR:** 이미지 속 글자를 디지털 텍스트로 변환하는 기술 [cite: 759]
* **MNIST:** 0~9 손글씨 이미지 7만 장의 학습 데이터셋 [cite: 760]

### 실습 키워드 [cite: 762]
* **npm:** Node.js의 패키지 관리자 [cite: 763]
* **배치 파일(.bat):** 여러 명령어를 한 번에 실행하는 Windows 스크립트 [cite: 764]
* **CLAUDE.md:** 프로젝트의 구조, 규칙, 명령어를 기록한 설정 파일 [cite: 765-766]
* **/init:** 폴더를 분석해 `CLAUDE.md`를 자동 생성하는 명령어 [cite: 686, 767]
* **#키:** 규칙을 `CLAUDE.md`에 빠르게 추가하는 지시어 [cite: 768]
