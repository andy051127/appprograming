# PRD Step 1 - 이미지 업로드 & 재료 인식

## 개요
사용자가 냉장고 사진을 업로드하면, AI가 이미지를 분석하여 식재료 목록을 자동으로 추출하는 기능

---

## 목표
- 사용자가 냉장고 사진을 쉽게 업로드할 수 있는 UI 제공
- OpenRouter API (nvidia/nemotron-nano-12b-v2-vl:free)를 이용해 이미지에서 재료 인식
- 인식된 재료를 목록 형태로 화면에 출력
- 재료 수동 수정(추가/삭제) 기능 제공

---

## 사용자 스토리
- 사용자는 냉장고 사진을 드래그&드롭 또는 파일 선택으로 업로드할 수 있다
- 사용자는 업로드한 이미지 미리보기를 확인할 수 있다
- 사용자는 AI가 분석한 재료 목록을 확인할 수 있다
- 사용자는 잘못 인식된 재료를 삭제하거나 빠진 재료를 직접 추가할 수 있다

---

## 기술 스택
| 구분 | 기술 |
|------|------|
| Frontend | HTML, CSS, JavaScript (Vanilla 또는 React) |
| Backend | Python Flask |
| AI API | OpenRouter API - `nvidia/nemotron-nano-12b-v2-vl:free` |
| 이미지 처리 | Base64 인코딩 후 API 전송 |

---

## API 명세

### POST /api/analyze
냉장고 이미지를 분석하여 재료 목록 반환

**Request**
```json
{
  "image": "data:image/jpeg;base64,..."
}
```

**Response**
```json
{
  "ingredients": ["달걀", "당근", "두부", "우유", "양파"]
}
```

**Error Response**
```json
{
  "error": "이미지 분석에 실패했습니다.",
  "code": 500
}
```

## 기능 상세

### 1. 이미지 업로드
- 지원 형식: JPG, JPEG, PNG, WEBP
- 최대 파일 크기: 10MB
- 업로드 방식: 파일 선택 다이얼로그 또는 드래그&드롭
- 업로드 즉시 미리보기 표시

### 2. 재료 인식 (AI 분석)
- 업로드된 이미지를 Base64로 인코딩
- OpenRouter API에 멀티모달 요청 전송
- 프롬프트: "이 냉장고 사진에서 보이는 모든 식재료를 한국어로 목록으로 나열해줘. JSON 배열 형태로 반환해줘."
- 응답에서 재료 목록 파싱

### 3. 재료 목록 수정
- 각 재료 옆 [x] 버튼으로 삭제
- 입력창으로 재료 직접 추가
- 수정된 재료 목록은 Step 2 (레시피 추천)에 전달
