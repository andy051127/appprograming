# PRD Step 2 - 레시피 추천

## 개요
Step 1에서 인식된 재료 목록을 기반으로, AI가 만들 수 있는 레시피를 추천하고 상세 조리법을 제공하는 기능

---

## 목표
- 인식된 재료로 만들 수 있는 레시피 3~5개 추천
- 각 레시피의 조리 시간, 난이도, 필요 재료 표시
- 선택한 레시피의 단계별 조리법 제공
- OpenRouter 텍스트 생성 모델 활용

---

## 사용자 스토리
- 사용자는 재료 목록을 확인한 후 "레시피 추천받기" 버튼을 클릭할 수 있다
- 사용자는 AI가 추천한 레시피 카드 목록을 볼 수 있다
- 사용자는 관심 있는 레시피를 클릭하여 상세 조리법을 확인할 수 있다
- 사용자는 부족한 재료가 무엇인지 확인할 수 있다

---

## 기술 스택
| 구분 | 기술 |
|------|------|
| Frontend | HTML, CSS, JavaScript |
| Backend | Python Flask |
| AI API | OpenRouter API - `google/gemma-4-27b-it:free` (텍스트 생성) |
| 데이터 형식 | JSON (레시피 구조화 응답) |

## API 명세

### POST /api/recipes
재료 목록을 받아 레시피 추천 목록 반환

**Request**
```json
{
  "ingredients": ["달걀", "당근", "두부", "양파"]
}
```

**Response**
```json
{
  "recipes": [
    {
      "id": 1,
      "name": "두부 달걀 볶음",
      "cooking_time": "15분",
      "difficulty": "쉬움",
      "available_ingredients": ["달걀", "두부", "양파"],
      "missing_ingredients": ["간장", "참기름"],
      "description": "간단하고 영양가 높은 두부 달걀 볶음입니다."
    },
    {
      "id": 2,
      "name": "당근 달걀 볶음밥",
      "cooking_time": "20분",
      "difficulty": "보통",
      "available_ingredients": ["달걀", "당근", "양파"],
      "missing_ingredients": ["밥", "간장"],
      "description": "남은 재료로 만드는 간편 볶음밥입니다."
    }
  ]
}
```

### POST /api/recipe/detail
특정 레시피의 단계별 조리법 반환

**Request**
```json
{
  "recipe_name": "두부 달걀 볶음",
  "ingredients": ["달걀", "두부", "양파"]
}
```

**Response**
```json
{
  "recipe_name": "두부 달걀 볶음",
  "steps": [
    "두부를 먹기 좋은 크기로 자른다.",
    "양파를 채 썰어 준비한다.",
    "팬에 기름을 두르고 양파를 볶는다.",
    "두부를 넣고 중불에서 3분간 볶는다.",
    "달걀을 풀어 넣고 익을 때까지 볶는다.",
    "소금, 후추로 간을 맞춘다."
  ],
  "tips": "두부는 키친타올로 물기를 제거하면 더 바삭하게 볶을 수 있습니다."
}
```

## 기능 상세

### 1. 레시피 추천
- Step 1의 재료 목록을 API에 전달
- 프롬프트 예시:
  ```
  다음 재료로 만들 수 있는 한국 요리 레시피 3가지를 추천해줘.
  재료: 달걀, 당근, 두부, 양파
  각 레시피는 이름, 조리시간, 난이도(쉬움/보통/어려움),
  사용 가능한 재료, 부족한 재료, 간단한 설명을 포함해서
  JSON 형식으로 반환해줘.
  ```
- 재료 일치율 높은 순으로 정렬하여 표시

### 2. 레시피 카드
- 요리 이름
- 조리 시간 (예: 15분)
- 난이도 (쉬움 / 보통 / 어려움)
- 보유 재료 / 전체 재료 비율 표시
- 부족한 재료 목록 표시

### 3. 조리법 상세
- 단계별 번호 매긴 조리 순서
- 요리 팁 (있을 경우)
- "즐겨찾기 추가" 버튼 (Step 3 연동)
