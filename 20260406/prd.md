# PRD: 상식 퀴즈 게임

## 1. 개요

### 1.1 제품 목적
사용자가 한국사, 과학, 지리, 일반상식 분야의 지식을 테스트하고 학습할 수 있는 4지선다 객관식 퀴즈 게임

### 1.2 대상 사용자

#### 페르소나 1 — 수능 준비생 김민준 (19세, 고3)
- **목표**: 한국사·사회 과목 암기 내용을 퀴즈로 반복 학습
- **행동 패턴**: 하루 30분 내외, 스마트폰으로 이동 중 또는 자투리 시간 활용
- **불편함**: 교재 반복 읽기는 지루함 → 게임 형식으로 흥미 유지 원함
- **기대**: 틀린 문제의 해설을 바로 확인해 약점 파악

#### 페르소나 2 — 취업 준비생 이수연 (26세, 대졸)
- **목표**: NCS·공기업 상식 시험 대비, 전반적인 일반상식 점검
- **행동 패턴**: PC와 모바일 병행, 하루 1~2 카테고리씩 집중 플레이
- **불편함**: 시중 앱은 광고가 많고 문제 수가 적음
- **기대**: 카테고리별 성취도 확인 → 취약 영역 집중 복습

#### 페르소나 3 — 직장인 박성호 (34세, 회사원)
- **목표**: 두뇌 자극 및 일상적 지식 확장을 위한 가벼운 게임
- **행동 패턴**: 점심시간·퇴근 후 10~15분, 주로 모바일
- **불편함**: 복잡한 회원가입·설치 과정 없이 바로 즐기고 싶음
- **기대**: 순위 경쟁으로 성취감, 지인과 점수 비교

## 2. 기능 요구사항

### 2.1 퀴즈 구성

| 항목 | 내용 |
|------|------|
| 문제 형식 | 4지선다 객관식 |
| 카테고리 | 한국사, 과학, 지리, 일반상식 |
| 카테고리별 문제 수 | 10문제 |
| 총 문제 수 | 40문제 |

#### 카테고리 상세 및 세부 출제 범위

**한국사** (10문제)

|----------|---------|------|

| 근현대 (개화기~현대) | 3문제 | 일제강점기, 광복, 6.25전쟁 |

| 서브토픽 | 문제 수 | 예시 |
|----------|---------|------|
| 물리 | 3문제 | 뉴턴 법칙, 전기·자기 |
| 화학 | 2문제 | 원소 주기율표, 산·염기 |
| 생물 | 3문제 | 세포 구조, 유전, 진화 |
| 지구과학 | 2문제 | 판 구조론, 기후 변화 |

**지리** (10문제)
| 서브토픽 | 문제 수 | 예시 |
|----------|---------|------|
| 세계 수도 | 3문제 | 각국 수도 위치 |
| 지형·자연환경 | 3문제 | 대륙·해양·산맥·강 |
| 국가·문화권 | 2문제 | 대륙별 국가, 언어권 |

**일반상식** (10문제)
| 서브토픽 | 문제 수 | 예시 |
|----------|---------|------|
| 사회·경제 | 3문제 | 기본 경제 용어, 사회 제도 |
| 문화·예술 | 2문제 | 세계 문화유산, 유명 예술 작품 |
| 스포츠 | 2문제 | 올림픽, 스포츠 규칙 |
| 시사·과학기술 | 3문제 | 최신 이슈, IT 용어 |

#### 난이도 시스템

**3단계 난이도 정의**
| 난이도 | 비율 | 설명 | 예시 |
|--------|------|------|------|
| 초급 (하) | 40% (4문제) | 교과서 기본 개념, 널리 알려진 사실 | "한국의 수도는?" |
| 중급 (중) | 40% (4문제) | 심화 개념, 혼동하기 쉬운 내용 | "임진왜란이 발생한 연도는?" |
| 고급 (상) | 20% (2문제) | 세부 사실, 응용·추론 필요 | "조선 세종 때 창제된 문자의 초성 수는?" |

**난이도별 점수 차등**
| 난이도 | 정답 점수 | 오답 점수 |
|--------|-----------|-----------|
| 초급 | +5점 | 0점 |
| 중급 | +10점 | 0점 |
| 고급 | +15점 | 0점 |

> 카테고리별 최고 점수: (4×5) + (4×10) + (2×15) = **90점**
> 전체 모드 최고 점수: **360점**

**난이도 선택**
- 게임 시작 전 난이도 선택 가능: 초급만 / 혼합(기본) / 고급만
- 기본값: 혼합 (초급 40% + 중급 40% + 고급 20%)

#### 문제 출제 방식 및 랜덤 로직

**출제 순서 규칙**
- 카테고리 모드: 해당 카테고리 내 서브토픽 순환 방식으로 출제 (한 서브토픽이 연속 2회 이상 나오지 않도록 제한)
- 전체 모드: 4개 카테고리를 라운드 로빈(round-robin) 방식으로 순환 출제 (한국사 → 과학 → 지리 → 일반상식 → 반복)

**랜덤 처리 정책**
- 세션 시작 시 문제 풀(pool)을 Fisher-Yates 알고리즘으로 셔플
- 동일 세션 내 같은 문제 중복 출제 없음
- 선택지 순서도 매 세션마다 랜덤 셔플 (정답 위치 고정 방지)
- 재시작 시 이전 세션과 다른 문제 순서 보장

### 2.2 게임 플로우

```
[시작 화면] ──────────────────────────────────────────────
  │  ├─> 카테고리 선택 + 난이도 선택
  │  └─> 순위 기록 보기 ──> [순위 화면] ──> 뒤로가기
  ▼
[문제 화면] ─── 상태: PLAYING
  │  └─> 선택지 클릭 (단 1회, 이후 입력 잠금)
  ▼
[피드백 오버레이] ─── 상태: FEEDBACK (문제 화면 위에 오버레이)
  │  ├─> 정답: 초록 배경 + 정답 메시지 + 해설
  │  └─> 오답: 빨간 배경 + 정답 표시 + 해설
  │  └─> "다음 문제" 버튼 클릭
  ▼
[문제 화면] (다음 문제) ── 마지막 문제 완료 시 ──>
  ▼
[결과 화면] ─── 상태: RESULT
  │  ├─> 닉네임 입력 후 저장 ──> [순위 화면]
  │  ├─> 다시 하기 ──> [문제 화면] (동일 카테고리/난이도)
  │  └─> 카테고리 변경 ──> [시작 화면]
  ▼
[순위 화면] ─── 상태: LEADERBOARD
       └─> 홈으로 ──> [시작 화면]
```

**예외 상황 처리**
| 상황 | 처리 방식 |
|------|-----------|
| 문제 화면에서 뒤로가기(브라우저) | "게임을 종료하시겠습니까?" 확인 모달 → 확인 시 시작 화면으로, 취소 시 문제 화면 유지 |
| 문제 화면에서 새로고침 | 동일 확인 모달 표시 (beforeunload 이벤트) |
| 피드백 중 뒤로가기 | 뒤로가기 차단, 문제 화면으로 복귀 |
| localStorage 용량 초과 | 가장 오래된 순위 1개 삭제 후 재저장, 사용자에게 토스트 알림 |

### 2.4 상태 관리

**게임 전체 상태 정의**

| 상태 | 진입 조건 | 허용 액션 | 차단 액션 |
|------|-----------|-----------|-----------|
| `IDLE` | 앱 최초 로드, 홈 복귀 | 카테고리 선택, 난이도 선택, 순위 보기 | 없음 |
| `PLAYING` | 카테고리/난이도 선택 후 게임 시작 | 선택지 클릭 1회 | 선택지 중복 클릭, 뒤로가기(모달 처리) |
| `FEEDBACK` | 선택지 클릭 직후 | "다음 문제" 버튼 클릭 | 선택지 재클릭, 뒤로가기 |
| `RESULT` | 마지막 문제 피드백 확인 후 | 닉네임 입력, 저장, 다시하기, 카테고리 변경 | 없음 |
| `LEADERBOARD` | 순위 기록 버튼 클릭 | 필터 탭 전환, 홈으로 이동 | 없음 |

**게임 세션 데이터 (메모리 내 유지)**
```
GameSession {
  category: string         // 선택된 카테고리 ('전체' 포함)
  difficulty: string       // 'easy' | 'mixed' | 'hard'
  questions: Question[]    // 셔플된 문제 배열
  currentIndex: number     // 현재 문제 인덱스
  score: number            // 누적 점수
  answers: {               // 문제별 답변 기록
    questionId: number
    selected: number       // 사용자가 선택한 인덱스
    correct: boolean
  }[]
}
```

## 4. 데이터 구조

### 4.1 문제 데이터
```json
{
  "id": 1,
  "category": "한국사",
  "question": "문제 텍스트",
  "choices": ["선택지1", "선택지2", "선택지3", "선택지4"],
  "answer": 2,
  "explanation": "해설 텍스트"
```
